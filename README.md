## 🌟 Key Features

### 🤖 Smart AI Diagnostic Tool
Allow customers to describe their issues or upload photos of leaks and clogs. Our integrated **Gemini AI** provides:
- **Instant Diagnosis:** Identifies the probable cause of the plumbing issue.
- **Cost Estimation:** Generates a realistic price range for repairs.
- **Priority Level:** Detects emergencies (High/Medium/Low) automatically.
- **DIY Safety Check:** Tells the user if it's safe to fix themselves or requires a pro.

### 📈 Modern Service Landing Page
- **Premium UI/UX:** Built with React and Vanilla CSS (Glassmorphism & High-end aesthetics).
- **Responsive Design:** Optimized for mobile (where most plumbing leads come from).
- **Lead Generation:** Integrated contact forms that capture AI diagnostic data for faster dispatching.

### 🛠️ Developer-First Architecture
- **Vite-Powered:** Blazing fast development and builds.
- **TypeScript:** Type-safe development for reliable scaling.
- **Modular Components:** Easily swap out AI services or UI sections.
